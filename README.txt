# CodePhantom Portable

Instructions :
1. Placez vos fichiers `index.ts`, `vite.config.ts`, `.env`, etc. dans ce dossier.
2. La base de données SQLite est déjà incluse : ./data/codephantom.db
3. Double-cliquez sur `build.bat` pour compiler en .exe (nécessite `pkg`)
4. Lancez ensuite `CodePhantom.exe`

Bonne programmation avec l'IA !
